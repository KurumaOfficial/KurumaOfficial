## Как использовать

1. Создай репо с именем == твой GitHub username (`Kuruma/Kuruma`)
2. Скопируй выбранный `vN_README.md` → `README.md`
3. Если у варианта есть SVG хедер — создай папку `assets/` и положи `vN_header.svg` → `assets/header.svg`
4. Замени `Kuruma` на реальный GitHub username во всех URL
5. Замени ссылки на telegram, email, сайт
6. Замени `repo=wettea` на реальное имя репо в pin-карточках
